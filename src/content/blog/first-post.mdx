# Getting Started with Astro and Tailwind CSS

![Astro and Tailwind CSS logos](@/assets/images/blog/tailwind-tips.png)

Astro is a modern static site builder that delivers lightning-fast performance with zero JavaScript by default. Combined with Tailwind CSS, you can build beautiful, responsive websites in no time.

## Why Astro?

Astro stands out from other frameworks because of its **island architecture**. This means you can use your favorite framework (React, Vue, Svelte, etc.) while still shipping minimal JavaScript.

### Key Features

- **Zero JavaScript by default** - Only ship what you need
- **Component islands** - Interactive components in a sea of static HTML
- **Partial hydration** - Hydrate only what's necessary
- **Build optimizations** - Automatic code splitting and tree shaking

## Setting Up Tailwind CSS

```bash
npm create astro@latest
cd my-astro-project
npx astro add tailwind

Creating Your First Component
Here's a simple component example:

---
// src/components/Button.astro
interface Props {
  variant?: 'primary' | 'secondary';
}

const { variant = 'primary' } = Astro.props;
---

<button class:list={[
  'px-4 py-2 rounded font-mono text-sm',
  variant === 'primary' 
    ? 'bg-zinc-900 text-white'
    : 'border border-zinc-200 text-zinc-700'
]}>
  <slot />
</button>

Conclusion
Astro + Tailwind CSS is a powerful combination for building fast, modern websites. The developer experience is excellent, and the performance is unmatched.
