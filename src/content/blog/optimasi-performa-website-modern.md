## Mengapa Performa Website Penting?

Kecepatan website berpengaruh langsung terhadap pengalaman pengguna. Website yang lambat dapat menyebabkan pengunjung meninggalkan halaman sebelum selesai dimuat.

Selain itu, performa juga memengaruhi peringkat mesin pencari.

## Teknik Optimasi Dasar

Berikut beberapa teknik optimasi yang umum digunakan:

### Lazy Loading

Lazy loading memungkinkan gambar dimuat hanya saat diperlukan.

Manfaatnya:

- Mengurangi beban awal halaman  
- Mempercepat waktu render  
- Menghemat bandwidth  

### Caching

Caching menyimpan data yang sering digunakan sehingga tidak perlu dimuat ulang setiap kali halaman diakses.

Jenis caching:

- Browser cache  
- CDN cache  
- Server cache  

### Code Splitting

Code splitting membagi kode JavaScript menjadi beberapa bagian kecil.

Tujuannya:

- Mengurangi ukuran bundle awal  
- Mempercepat loading halaman  

## Kesimpulan

Optimasi performa bukan hanya tentang kecepatan, tetapi juga tentang pengalaman pengguna. Website yang cepat akan meningkatkan kepuasan pengguna dan konversi.
