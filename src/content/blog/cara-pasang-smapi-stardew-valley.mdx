Main **Stardew Valley** di Android memang seru. Tapi kalau sudah mulai bosan dengan versi vanilla, biasanya langsung kepikiran buat pasang mod.

Nah, supaya mod bisa berjalan di Android, kamu wajib memasang **SMAPI** (*Stardew Modding API*).

Tanpa SMAPI, mod apa pun yang kamu unduh tidak akan terbaca di game. Sayangnya, proses instalasinya memang sedikit berbeda dibanding versi PC, dan sering kali gagal hanya karena ada langkah kecil yang terlewat.

![SMAPI Stardew Valley Android](https://cdn.c0desk1.my.id/img/AgACAgUAAxkBAAOEahLd-7_beMDMH2Oj0RVeu4BIsfwAAp0Paxv8f5hUcX0GUEbir0wBAAMCAAN3AAM7BA)

Di artikel ini saya akan pandu langkah demi langkah cara memasang SMAPI Stardew Valley Mobile dengan benar berdasarkan pengalaman langsung.

---

## Persiapan Sebelum Mulai
Sebelum mulai, pastikan dulu beberapa hal berikut:

- **Stardew Valley Mobile** sudah terpasang di HP kamu.
- Aplikasi **File Manager**.
- Aplikasi **SMAPI Launcher**.
- **Koneksi internet stabil** untuk mengunduh file.
- **Baterai cukup**, karena proses instalasi bisa memakan waktu sekitar 10–15 menit.

Kalau semua sudah siap, lanjut ke langkah berikut.

Kalau belum punya bahannya, tenang—link download saya sertakan di bawah.

### Unduh SMAPI untuk Android
Buka browser di HP kamu lalu kunjungi halaman resmi SMAPI di NexusMods.

Kamu perlu membuat akun NexusMods terlebih dahulu sebelum bisa mengunduh file.

Pastikan file yang kamu download berformat `.zip`, biasanya bernama seperti: `SMAPI-Android-x.x.x.x.zip` Saat artikel ini ditulis, versi Android terbaru yang bisa kamu gunakan adalah :badge[v4.3.2.5].

:::warning
Jangan mengunduh dari sumber yang tidak jelas. File mod dari sumber tidak resmi berisiko bermasalah atau mengandung malware.
:::

### Instal APK SMAPI Launcher
Setelah file berhasil diunduh:

- **Buka File Manager**
- Cari file APK-nya di dalam folder Download.
- Tekan file APK
- Pilih **Install**

:::filetree
- storage/emulated/0/
  - Android/
  - Documents/
  - Download/
    - SMAPI-Android-x.x.x.x.zip
    - **SMAPILauncher.apk**
    - ~~kenangan-mantan.mp4~~
    - ~~pap-dari-mantan.jpg~~
:::

Kalau muncul peringatan *Unknown apps* atau *Sumber tidak dikenal*:

- Buka **Settings**
- Pilih **Apps**
- Masuk ke **Special access**
- Pilih **Install unknown apps**
- Aktifkan izin untuk browser atau file manager yang kamu pakai

### Buka Aplikasi SMAPI Launcher
Setelah terpasang:

- Buka **SMAPI Launcher**
- Pilih **Install SMAPI from Zip**
- Cari file ZIP SMAPI yang tadi kamu unduh

:::filetree
- storage/emulated/0/
  - Android/
  - Documents/
  - Download/
    - **SMAPI-Android-x.x.x.x.zip**
    - SMAPILauncher.apk
    - ~~kenangan-mantan.mp4~~
    - ~~pap-dari-mantan.jpg~~
:::

Tunggu sampai proses instalasi selesai. Jangan keluar dari aplikasi selama proses berlangsung.

### Verifikasi Instalasi
Kalau instalasi sudah selesai:

- Pilih **Start Game**

Saat game terbuka, lihat pojok kiri bawah layar.

Kalau muncul tulisan seperti: **SMAPI x.x.x** berarti instalasi berhasil.

### Mulai Pasang Mod
Setelah SMAPI aktif:

- Buka **Mod Manager**
- Pilih **Install Mod**
- Cari mod yang ingin dipasang.

:::tip
Beberapa mod membutuhkan dependensi tambahan seperti **Content Patcher** atau utilitas dari SMAPI. Pastikan baca instruksi di halaman mod sebelum memasang.
:::

## Troubleshooting Umum
| Masalah | Solusi |
|---------|--------|
| SMAPI Launcher tidak muncul di layar utama | Restart HP |
| Game crash setelah mod dipasang | Hapus mod satu per satu untuk cari yang bermasalah |
| Tidak bisa akses folder `Android/data` | Gunakan **SMAPI Launcher** lalu buka **Mod Manager** |
| Error "SMAPI not installed" | Ulangi proses instalasi dari ZIP |
| File Apk berformat **APKS**/**XAPK** | Gunakan Aplikasi XAPK Installer |
