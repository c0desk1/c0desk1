# Portfolio Website 2024

A complete redesign of my personal portfolio with a focus on industrial tactical aesthetics and performance.

## The Challenge

Creating a portfolio that stands out while maintaining excellent performance and accessibility. The goal was to achieve a unique "industrial tactical" vibe without sacrificing user experience.

## The Solution

Built with Astro for static generation and Tailwind CSS for styling. The site features:

- **Industrial tactical design** with monospace typography and subtle grid patterns
- **Dark mode support** with system preference detection
- **MDX integration** for blog posts and case studies
- **Automatic sitemap** and RSS feed generation
- **Optimized images** with lazy loading

## Technical Implementation

### Architecture

The site uses Astro's content collections for type-safe blog and portfolio management. All content is written in MDX, allowing for interactive components within articles.

### Performance Optimizations

- Images are automatically optimized and converted to WebP
- CSS is purged and minified
- JavaScript is minimal and only loaded when needed
- Critical CSS is inlined for above-the-fold content

### Accessibility Features

- Proper semantic HTML structure
- ARIA labels where necessary
- Keyboard navigation support
- Focus indicators for interactive elements

## The Result

The new portfolio achieves 98+ scores on Lighthouse across all metrics while maintaining a unique visual identity that reflects my personal brand.
