**c0desk1**ウェブサイトは、ユーザー体験の向上、トラフィック分析、広告表示のためにクッキーを使用しています。

---

## 1. クッキーとは？

クッキーは、ウェブサイトを訪問した際にお客様のデバイス（コンピューター、スマートフォン、タブレット）に保存される小さなテキストファイルです。クッキーはウェブサイトが以下を行うのを支援します：

- お客様の設定を記憶すること
- お客様のウェブサイトの利用状況を把握すること
- 関連性の高いコンテンツを表示すること

## 2. 当方が使用するクッキー

| クッキーの種類 | 機能 | 期間 |
| -------------------- | -------------------------------------------------------------------------- | --------- |
| **必須クッキー** | テーマ設定（ダーク/ライトモード）の保存 | 1年 |
| **分析クッキー** | トラフィックと訪問者の行動の測定（Google Analytics、Yandex Metrica） | 2年 |
| **広告クッキー** | 関連性の高い広告の表示（Google AdSense、Yandex Ads） | 1〜2年 |

## 3. サードパーティクッキー

当方はクッキーを配置するサードパーティサービスを利用しています：

| サービス | 機能 | ポリシー |
| -------------------- | --------------- | -------------------------------------------------------------------- |
| **Google Analytics** | トラフィック分析 | [Googleポリシー](https://policies.google.com/technologies/cookies) |
| **Yandex Metrica** | トラフィック分析 | [Yandexポリシー](https://yandex.com/legal/cookies_policy/) |
| **Google AdSense** | 広告 | [Googleポリシー](https://policies.google.com/technologies/ads) |
| **Yandex Ads** | 広告 | [Yandexポリシー](https://yandex.com/legal/advertising/) |

## 4. クッキーの管理方法

ブラウザの設定を通じてクッキーを管理できます：

| ブラウザ | 方法 |
| ----------- | ----------------------------------------- |
| **Chrome** | 設定 → プライバシーとセキュリティ → クッキー |
| **Firefox** | オプション → プライバシーとセキュリティ → クッキー |
| **Safari** | 環境設定 → プライバシー → クッキー |
| **Edge** | 設定 → プライバシーとセキュリティ → クッキー |

クッキーを無効にすると、以下のようにウェブサイトの機能に影響が出る可能性があります：

- テーマ設定が保存されない
- 一部の機能が最適に動作しない場合がある

## 5. 広告クッキーと設定

以下のリンクから広告設定を管理できます：

- [Google広告設定](https://adssettings.google.com/)
- [Yandex広告設定](https://yandex.com/support/ad-settings/)

## 6. クッキーポリシーの変更

当方はこのクッキーポリシーをいつでも更新することができます。変更は公開後に有効となります。
