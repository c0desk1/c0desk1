Kami menghargai privasi Anda. Kebijakan privasi ini menjelaskan bagaimana website **c0desk1** mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda.

---

## 1. Informasi yang Dikumpulkan

Kami dapat mengumpulkan informasi berikut:


| Jenis Data | Contoh | Cara Pengumpulan |
| ------------------- | ---------------------------------------- | ----------------------------- |
| **Data teknis** | Alamat IP, jenis browser, perangkat | Otomatis (via analytics) |
| **Data penggunaan** | Halaman yang dikunjungi, waktu kunjungan | Otomatis (via analytics) |
| **Data komunikasi** | Email, pesan, komentar | Formulir kontak atau komentar |
| **Cookie** | Preferensi pengguna | Otomatis (via browser) |


## 2. Penggunaan Informasi

Informasi yang kami kumpulkan digunakan untuk:

- Menyediakan dan mengoperasikan website
- Menganalisis performa dan penggunaan website
- Memperbaiki konten dan pengalaman pengguna
- Menanggapi pertanyaan atau komentar Anda
- Menampilkan iklan yang relevan (Google AdSense / Yandex Ads)

## 3. Pihak Ketiga

Kami menggunakan layanan pihak ketiga yang mungkin mengumpulkan data Anda:


| Layanan | Fungsi | Lokasi Data |
| -------------------- | ------------------ | ----------------- |
| **Google Analytics** | Analisis trafik | Server Google |
| **Yandex Metrica** | Analisis trafik | Server Yandex |
| **Google AdSense** | Iklan | Server Google |
| **Yandex Ads** | Iklan | Server Yandex |
| **Cloudflare** | CDN & keamanan | Server Cloudflare |
| **GitHub** | Hosting repository | Server GitHub |


Layanan pihak ketiga memiliki kebijakan privasi masing-masing. Kami sarankan Anda membacanya.

## 4. Cookie

Website ini menggunakan **cookie** untuk:

- Menyimpan preferensi tema (dark/light mode)
- Menganalisis trafik pengunjung
- Menampilkan iklan yang relevan

Anda dapat mengatur browser untuk menolak cookie. Namun, beberapa fitur website mungkin tidak berfungsi optimal.

## 5. Iklan

Website ini menampilkan iklan dari:

- **Google AdSense**
- **Yandex Ads**

Iklan ditampilkan berdasarkan konten dan perilaku pengguna. Kami tidak mengontrol iklan spesifik yang muncul.

Anda dapat mengatur preferensi iklan melalui:

- [Google Ad Settings](https://adssettings.google.com/)
- [Yandex Ad Settings](https://yandex.com/support/ad-settings/)

## 6. Keamanan Data

Kami mengambil langkah-langkah wajar untuk melindungi data Anda. Namun, **tidak ada metode transmisi data yang 100% aman**.

Kami tidak bertanggung jawab atas akses tidak sah oleh pihak ketiga di luar kendali kami.

## 7. Data Anak-Anak

Website kami tidak ditujukan untuk anak di bawah 13 tahun. Kami tidak secara sengaja mengumpulkan data pribadi dari anak-anak.

Jika Anda yakin kami memiliki data anak di bawah 13 tahun, harap hubungi kami.
