Dengan menggunakan **c0desk1**, Anda menyetujui praktik data yang dijelaskan dalam Kebijakan Privasi ini.

### 1. Data yang Kami Kumpulkan
Kami mengumpulkan data berikut untuk mengoperasikan dan meningkatkan layanan:

| Jenis Data | Contoh | Cara Pengumpulan |
| --- | --- | --- |
| **Data Teknis** | Alamat IP, jenis browser, perangkat | Otomatis melalui analytics |
| **Data Penggunaan** | Halaman yang dikunjungi, waktu kunjungan | Otomatis melalui analytics |
| **Data Komunikasi** | Email, pesan, komentar | Saat Anda mengisi formulir |
| **Cookie** | Preferensi tema, sesi pengguna | Otomatis melalui browser |

### 2. Cara Kami Menggunakan Data
Data digunakan untuk:
- Menyediakan dan memelihara website
- Menganalisis performa dan penggunaan
- Memperbaiki konten dan pengalaman pengguna
- Menanggapi pertanyaan Anda
- Menampilkan iklan relevan melalui Google AdSense dan Yandex Ads

### 3. Layanan Pihak Ketiga
Kami menggunakan layanan pihak ketiga yang mungkin memproses data Anda:

| Layanan | Fungsi | Lokasi Data |
| --- | --- | --- |
| **Google Analytics** | Analisis trafik | Server Google |
| **Yandex Metrica** | Analisis trafik | Server Yandex |
| **Google AdSense** | Iklan | Server Google |
| **Yandex Ads** | Iklan | Server Yandex |
| **Cloudflare** | CDN & keamanan | Server Cloudflare |

Setiap layanan memiliki kebijakan privasi masing-masing.

### 4. Cookie
Kami menggunakan cookie untuk menyimpan preferensi tema, menganalisis trafik, dan menampilkan iklan.

Anda dapat menonaktifkan cookie di pengaturan browser. Beberapa fitur mungkin tidak berfungsi jika cookie dimatikan.

### 5. Iklan
Iklan ditampilkan melalui Google AdSense dan Yandex Ads berdasarkan konten dan perilaku pengguna.

Atur preferensi iklan Anda di:
- [Google Ad Settings](https://adssettings.google.com/)
- [Yandex Ad Settings](https://yandex.com/support/ad-settings/)

### 6. Keamanan Data
Kami menerapkan langkah keamanan wajar untuk melindungi data Anda. 

DATA DITRANSMISIKAN DENGAN RISIKO ANDA SENDIRI. TIDAK ADA METODE TRANSMISI INTERNET YANG 100% AMAN.

### 7. Privasi Anak
Layanan ini tidak ditujukan untuk anak di bawah 13 tahun. Kami tidak sengaja mengumpulkan data anak di bawah 13 tahun.

Jika Anda yakin kami memiliki data tersebut, hubungi kami untuk menghapusnya.

### 8. Hak Anda
Tergantung lokasi Anda, Anda berhak:
- Mengakses data pribadi yang kami simpan
- Meminta koreksi atau penghapusan data
- Membatasi atau menolak pemrosesan data
- Menarik persetujuan kapan saja

Untuk mengajukan permintaan, hubungi kami melalui [Kontak](/contact/).

### 9. Perubahan Kebijakan
Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan berlaku segera setelah dipublikasikan.

### 10. Kontak
Untuk pertanyaan tentang kebijakan privasi ini, hubungi kami melalui halaman [Kontak](/contact/).
